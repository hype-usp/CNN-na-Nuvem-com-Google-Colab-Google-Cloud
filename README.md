# CNN-na-Nuvem-com-Google-Colab-Google-Cloud
O projeto aborda a aplicação de visão computacional utilizando redes neurais convolucionais (CNNs) para a classificação de imagens. A ideia central é desenvolver/treinar modelos em um ambiente de Cloud, usando ferramentas do Colab para treinamento do modelo e o Cloud para deploy. Essa abordagem é importante para verificar se é possível/viável realizar o desenvolvimento completo de soluções de Machine Learning sem a necessidade de uma infraestrutura local potente, além de compreender as vantagens e desvantagens da nuvem, em termos de estabilidade, confiabilidade, manutenção e etc.

#Objetivos:
- Treinar um modelo CNN para classificação binária de imagens.
- Utilizar recursos do Google Colab para processamento e treino.
- Preparar o modelo para deploy no Vertex AI (Google Cloud).
- Organizar uma estrutura clara para avançar às etapas de API e produto.

#Ferramentas Usadas:
- Python, TensorFlow/Keras
- TensorFlow Datasets (TFDS)
- Google Colab
- Google Cloud Platform (Cloud Storage, Vertex AI)
- GitHub para versionamento

#Etapas do desenvolvimento
- Estudo e referência: análise de projetos semelhantes e técnicas de CNN.
- Treinamento: criação do modelo, data augmentation e avaliação.
- Preparação do modelo: exportação e organização dos artefatos.
- Deploy (em progresso): envio do modelo para o Vertex AI.
- API (futuro): criação de endpoint para consumo das previsões.
